# Git Cutlass

Git Cutlass is a simple curses-like tool to work with git (Work in progress)

# Current features

- Add, reset, checkout files
- Make commits
- View diffs

# Install

```
npm install git-cutlass -g
```

# Run
In a git repository folder
```
gitc
```

# Node version

Git Cutlass requires node `v4.1.0` or more to run.
